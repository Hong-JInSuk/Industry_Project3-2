## [ Industry_Project3-2 (Project Name) ]

이번 프로젝트는 'PC의 실제 게임실행시간 실시간 모니터링'입니다. 게임이 백그라운드로 켜져있는 것이 아닌 실제로 플레이하는 시간을 측정하여 통계를 내줍니다.
더 나아가서 게임 플레이 통계에 따라서 최근 게임 사용시간의 증가와 감소에 따른 보고서도 보여줄 예정입니다.
